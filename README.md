# NER

This repository contains pipelines for named entity recognition, relation extraction, and downstream classification/grouping.

## Project Structure

Main entry points:

- **Named Entity Recognition (NER):** `New_Relation_Extraction/NER.py`
- **Relation Extraction:** `New_Relation_Extraction/relation_extraction.py`
- **Classification into Groups:** `New_Relation_Extraction/final_classification_pipeline.ipynb`
